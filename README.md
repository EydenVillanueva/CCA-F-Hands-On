# CCA-F-Hands-On

Repositorio de práctica para la preparación del examen de certificación **CCA-F**
(Claude Certified Associate - Fundamentals).

Contiene la implementación de proyectos hands-on utilizando el **SDK de Claude**
(Claude Agent SDK / Anthropic API).

## Objetivo

Practicar y consolidar los conceptos fundamentales necesarios para la
certificación mediante ejemplos ejecutables.

## Requisitos

- Python 3.10+
- Una API key de Anthropic (`ANTHROPIC_API_KEY`)

## Configuración

```bash
python -m venv .venv
source .venv/bin/activate
pip install anthropic
export ANTHROPIC_API_KEY="tu-api-key"
```
